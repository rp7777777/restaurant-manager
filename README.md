# 🍽️ Servora ERP

Restaurant Management System built with React Native, React, Firebase & Firestore.

## Features

- 📊 Dashboard
- 💰 Sales Management
- 💸 Expense Management
- 📦 Inventory
- 👨‍🍳 Kitchen Requests
- 🏪 Store Management
- 👥 Employee Management
- 🕒 Attendance
- 💵 Payroll
- 🔒 Role Based Authentication
- 📈 Reports & Analytics
- ☁️ Real-time Firestore Sync

## Tech Stack

- React Native (Expo)
- React + Vite
- TypeScript
- Firebase Authentication
- Cloud Firestore
- React Navigation

## Folder Structure

```
servora-mobile/
├── app/
├── components/
├── hooks/
├── services/
├── constants/
├── utils/
└── firebase.ts
```

## Installation

```bash
git clone https://github.com/rp7777777/restaurant-manager.git

cd restaurant-manager

npm install

npx expo start
```
